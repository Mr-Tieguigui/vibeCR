Four steps from zero to a fully managed research project — LangSkills-density, zero fluff.

📋 Templates & Formats

LabPilot is entirely API-key-free. You use Gemini, ChatGPT, or Claude in your browser to fill templates — then upload the result. No provider configuration ever needed.

Project Template (YAML → JSON)

12 structured sections: definition, motivation, scope, questions, concepts, pipeline, evaluation, deliverables, milestones, risks, literature, next actions.

Literature CSV Formats

Undermind CSV (relevance scores, topic match) and Zotero CSV (Item Type, Abstract Note, DOI). Auto-detected from column headers.

Literature Workflow

Two sources, one unified pipeline. Undermind for AI-powered discovery, Zotero for curated libraries.

🔬 Undermind

Source Cloud AI search (app.undermind.ai)
Best for Discovering new papers on a research topic
Export Download CSV from search results
Key fields Relevance Summary, Topic Match Score, Abstract
Unique AI relevance scoring and topic matching

📚 Zotero

Source Local reference manager (zotero.org)
Best for Managing a curated personal library
Export Select items → Export → CSV
Key fields Item Type, Abstract Note, DOI, Date Added
Unique Rich metadata, notes, and PDF annotations

Why both? Undermind excels at AI-powered paper discovery — finding relevant work you didn't know existed. Zotero excels at organizing papers you've already found. Use Undermind to explore, Zotero to curate, and LabPilot to connect both to your execution pipeline.

Features

Everything you need to manage research projects end-to-end.

Documentation

Core concepts, daily workflow, and local AI usage — condensed.

🗂 Core Objects

  • Project — research initiative with overview, steps, timeline, and artifacts
  • Paper — literature entry with metadata, tags, notes, and project links
  • Step — execution task with status cycling (todo → doing → done → blocked)
  • Artifact — deliverable: code repo, dataset, paper draft, demo, or report
  • Report — auto-generated daily/weekly markdown summary from project data

📅 Daily Workflow

  • Open dashboard → check active step statuses and blockers
  • Cycle step status as you work (todo → doing → done)
  • Log progress with timestamped punch-in entries
  • Import new papers from Undermind/Zotero as discovered
  • Generate daily report to capture the day's progress

📈 Weekly Workflow

  • Review portfolio dashboard for RAG status and blocker trends
  • Update milestone statuses and kanban board positions
  • Tag and annotate papers accumulated during the week
  • Register new artifacts (code, data, drafts) to the artifact index
  • Generate weekly report with portfolio-level progress metrics

🤖 Local Web AI Usage

  • Download the YAML template from any project page
  • Paste into browser-based AI (ChatGPT, Gemini, Claude) and ask it to fill all sections
  • Save the output as JSON, upload via the "Upload Filled Template" button
  • No API keys, no provider config, no server-side AI calls
  • All AI interaction happens in your browser — LabPilot only processes the structured result
